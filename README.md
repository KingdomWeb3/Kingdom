# Kingdom
（English version inside the repo）KingDOM 是首个将去中心化社交与边缘AI推理网络深度融合的平台。用户日常观看、互动时无感贡献闲置算力；平台聚合后为AI公司提供低成本推理服务。创作者通过观看者自动支付的代币获利，代币KING锚定真实算力需求，通过B端付费回购销毁实现通缩。边缘AI推理市场预计2030年达2700亿美元，KingDOM以社交为入口，聚合海量碎片化算力，为中小AI团队提供高性价比服务，代币经济自洽，飞轮效应可期，链上内容形式拓展前景良好，与AI Agents + Autonomous Economy等诸多项目未来亦存在联动。
